# Libraries

Place custom browser-side libraries here.

Files under this folder are served by the app at `/libraries/*`.

Example:

- `libraries/p5/p5.min.js` is available at `/libraries/p5/p5.min.js`
- `libraries/createjs/createjs.min.js` is available at `/libraries/createjs/createjs.min.js`

This folder is intended for vendored libraries that should be loaded directly by the control or output HTML shells.
